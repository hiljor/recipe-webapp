# MVP User Stories

## 1. As a user, I want to see a list of all recipes at /recipes so that I can browse them.
* Acceptance criteria:
  - Given I am a new user, when I navigate to /recipes, then I should see a list of all recipes.

## 2. As a user, I want to be able to click on a recipe in the list at /recipes and be taken to the recipe details page at /recipes/:id.
* Acceptance criteria:
  - Given I am a new user, when I navigate to /recipes and click on a recipe, then I should be taken to the recipe details page at /recipes/:id.

## 3. As a user, I want to see the details of a recipe at /recipes/:id so that I can learn how to make it.
* Acceptance criteria:
  - Given I am a new user, when I navigate to /recipes/:id, then I should see the details of the recipe with the corresponding id.

## 4. As a user, I want to be able to register for an account at /register so that I can log in and access additional features.
* Acceptance criteria:
  - Given I am a new user, when I navigate to /register and fill out the registration form, then I should be able to create an account.

## 5. As a registered user, I want to be able to log in at /login so that I can access my account and additional features.
* Acceptance criteria:
  - Given I am a registered user, when I navigate to /login and enter my credentials, then I should be able to log in to my account.
